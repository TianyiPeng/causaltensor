Welcome to CausalTensor's documentation!
========================================

**CausalTensor** is a Python library for doing causal inference and policy evaluation using panel data.
It addresses questions like "What is the impact of strategy X to outcome Y" given time-series data colleting from multiple units. 
Answering such questions has wide range of applications from econometrics, operations research, business analytics, polictical science, to healthcare. 

Check out the :doc:`usage` section for further information.

.. note::

   This project is under active development.
